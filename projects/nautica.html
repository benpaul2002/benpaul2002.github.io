Overview

Nautica is a 3D game developed with Three.js, along with its trailer. The game offers players an immersive experience of travelling treacherous waters to collect treasures and battle against enemy ships. The project demonstrates full-stack 3D web development and game design in the browser.

Features

  • Immersive 3D naval exploration and combat
  • Treasure collection across treacherous waters
  • Enemy ship battles
  • Custom game trailer produced for the project

Tech Stack

  • Three.js
  • JavaScript
  • WebGL
  • 3D
