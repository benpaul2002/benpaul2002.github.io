Another Day

Retro 3D survival horror game built in Unity with a team of 3. AI-driven enemies, full storyline, cutscenes, and intense combat with custom-animated weapons.

Overview

Another Day is a retro-style 3D survival horror game developed in Unity as part of a 3-person team. The game combines classic horror atmosphere with modern gameplay systems: AI enemies that navigate the environment using Unity's NavMesh, a full narrative with multiple cutscenes and a dialogue system, and combat mechanics featuring multiple weapons, each with custom animations.

Features

  • AI enemy navigation using Unity's NavMesh system for challenging encounters
  • Comprehensive storyline with multiple cutscenes
  • Full dialogue system for narrative delivery
  • Intense combat mechanics with multiple weapons
  • Custom animations for each weapon

Tech Stack

  • Unity
  • C#
  • NavMesh
  • 3D Game Dev
