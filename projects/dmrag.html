DMRAG

Dungeon Master Retrieval-Augmented Generation system that automates the Dungeon Master role in Dungeons & Dragons.

View on GitHub

Overview

DMRAG is a Retrieval-Augmented Generation (RAG) system designed to automate the Dungeon Master role in Dungeons & Dragons. The system delivers immersive, context-aware narratives for players by combining fine-tuned language models with hybrid retrieval mechanisms.

The project demonstrates advanced NLP techniques, including fine-tuning transformers on domain-specific data, implementing hybrid retrieval strategies, and using OpenAI function calling for game state management.

Features

  • Fine-tuned sentence transformer for semantic search
  • GPT2 model trained on D&D-specific data
  • Hybrid retrieval combining BM25 (keyword-based) and semantic similarity
  • OpenAI function calling for dynamic game state management
  • Modular pipeline architecture for easy extension
  • Context-aware narrative generation based on game history

Tech Stack

  • Python
  • PyTorch
  • Transformers
  • OpenAI API
  • Sentence Transformers
  • BM25
  • RAG
  • NLP
